# Soul of Symphony — v2 React Website

High-end, Webflow-style React website. Dark luxury aesthetic.

## QUICK START (Localhost)

```bash
cd soul-of-symphony
npm install
npm run dev
```
Open: http://localhost:5173

## VERCEL DEPLOY

```bash
npm install -g vercel
vercel login
vercel
```

Live in 2 minutes at https://your-site.vercel.app
